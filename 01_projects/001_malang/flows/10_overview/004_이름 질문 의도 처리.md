# 이름 질문 의도 처리

## 목적
- "이름이 뭐에요?" 질문에 제품 의도에 맞는 명확한 답변을 일관되게 반환한다.

## 진입 조건
- 사용자 입력에 이름/호칭 관련 의도가 포함된다.

## 메인 플로우
```mermaid
flowchart TD
  A[사용자 질문 수신] --> B[의도 분류: 이름/호칭]
  B -->|이름 질문| C[브랜드/캐릭터 정책 조회]
  B -->|기타 의도| X[일반 대화 플로우로 전달]
  C --> D[허용 응답 템플릿 선택]
  D --> E[문맥 맞춤 후 응답 생성]
  E --> F[응답 반환 및 로그 기록]
```

## 예외 분기
- 정책 조회 실패 -> 기본 안전 템플릿으로 응답한다.
- 의도 분류 불확실 -> 확인 질문 후 재분류한다.
